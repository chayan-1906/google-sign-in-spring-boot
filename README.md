## Spring Boot Google Sign In

Demo Google Sign In Application built on SpringBoot
<br>
[![](https://i.ytimg.com/vi/qcz2jBLNOtc/hqdefault.jpg?sqp=-oaymwEcCNACELwBSFXyq4qpAw4IARUAAIhCGAFwAcABBg==&rs=AOn4CLDjJEzCV4Nj_0x59-xhi8jwL6rxsw)](https://youtu.be/qcz2jBLNOtc?list=PLVz2XdJiJQxxAsY2MfcY5acpNWMhgYV3)

The server runs on Tomcat, on <http://localhost:8090/home>

## Dependencies
* [Spring Boot Starter Oauth2 Client](https://mvnrepository.com/artifact/org.springframework.boot/spring-boot-starter-oauth2-client)
* [Spring Boot Starter Web](https://mvnrepository.com/artifact/org.springframework.boot/spring-boot-starter-web)
* [Spring Boot Starter Test](https://mvnrepository.com/artifact/org.springframework.boot/spring-boot-starter-test)
* [Javax Servlet API](https://mvnrepository.com/artifact/javax.servlet/servlet-api)
